# CRUD de Cadastro de Produtos em Java

Projeto desenvolvido em **Java puro** com o objetivo de praticar
lógica de programação, orientação a objetos e operações básicas de CRUD
(Create, Read, Update, Delete).

O sistema permite o cadastro e gerenciamento de produtos diretamente
pelo console, sem uso de banco de dados, utilizando estruturas em memória.

## Funcionalidades
- Cadastrar produtos
- Listar produtos cadastrados
- Atualizar informações de um produto
- Remover produtos pelo ID

## Como executar o projeto
1. Clone o repositório
2. Abra o projeto em uma IDE Java (Eclipse, IntelliJ ou VS Code)
3. Compile o projeto
4. Execute a classe `Program`

## Tecnologias utilizadas
- Java
- Git
- GitHub

## Objetivo do projeto
Este projeto foi criado com fins educacionais, focando no aprendizado
de Java puro e na construção de uma base sólida para projetos mais
avançados no futuro.
